underscore.js的基础知识1----数组操作方法

数组Array

1. _.first(array，[n])返回数组第一个元素,n表示返回数组中前 n 个元素.

2. _.last(array,[n])返回数组最后一个元素,n表示返回数组中后 n 个元素.

3. _.initial(array,[n])返回数组中除了最后一个元素之外的数组,n表示从最后一个开始向前排除的元素个数

4. _.rest(array,[n])返回数组中除了第一个元素之外的数组,n表示返回从第n个开始剩余的元素

5. _.flatten(array,shallow) 将一个嵌套多层的数组 array（数组）转换为只有一层的数组。 如果你传递 shallow参数位true，数组将只减少一维的嵌套。

6. _.without(array,*values) 返回一个删除所有values值后的 array副本

7. _.union(*arrays) 返回传入的 arrays（数组）并集：按顺序返回，返回数组的元素是唯一的

8. _.intersection(*arrays) 用数组形式返回arrays中共同的元素

9. _.difference(array,*others) 但返回的值来自array参数数组，并且不存在于other 数组。

10. _.uniq(array)去重，返回去重之后的数组

11. _.zip(*arrays) 将每个 arrays 中相应位置的值合并在一起

12. .unzip(array)与.zip(*arrays)相反，其第一元素个包含所有的输入数组的第一元素，其第二包含了所有的第二元素，依此类推。

13. _.object(list,[values]) 将数组转换为对象,传递任何一个单独[key, value]对的列表，或者一个键的列表和一个值得列表

14. _.chunk(array,length) 将 array 分成多个数组，每个数组包含length 或更少的项

15. _.indexOf(array,value) 返回在array中值为value的索引值(index)

16. _.lastIndexOf(array,value, [fromIndex])返回value在该 array 中的从最后开始的索引值，如果value不存在 array中就返回-1,传递fromIndex将从你给定的索性值开始搜索。

17. _.sortedIndex(array,value, [iteratee], [context]) 二分查找方法进行查找value,如果提供 iteratee 函数，iterator将作为list排序的依据，包括你传递的value 。iterator也可以是字符串的属性名用来排序(比如length)

18. .findIndex(array,predicate,[content]) 类似于.indexOf，当predicate通过真检查时，返回第一个索引值；否则返回-1。

19. .findLastIndex(array,predicate,[content]) 和.findIndex类似，但反向迭代数组，当predicate通过真检查时，最接近末端的索引值将被返回。

20. _.range([start], stop, [step]) 一个用来创建整数灵活编号的列表的函数，便于each 和 map循环
